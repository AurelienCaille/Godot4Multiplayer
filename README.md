# Godot4Multiplayer
/!\ Very early stage

Godot 4 Addon for multiplayer

Uses:
- with Nakama
- IP direct

This addon include 2 parts:
- Basics UIs for multiplayer connections
- MultiplayerManager singleton to manage both Nakama/IP direct
